Recurring tasks that work below the day level

TimeboardApp

A lightweight, dockerized task board built for schedules like “every 8 hours after completion”, “at 08:00/15:00/23:00”, and fixed clock intervals. Mobile-friendly UI, SQLite storage, and a full OpenAPI-documented API.

Recurrence

Sub‑daily intervals

Post-completion intervals (e.g. every 8h after you check it off) and fixed-clock schedules (anchored to the due time).

UI

Mobile + desktop

Responsive UI with theme support, calendar view, filters, and per-user preferences.

API

Documented

OpenAPI/Swagger documentation included. Automate tasks, notifications, and integrations.

Quick start (Docker)

Run locally with Docker Compose. Defaults to http://localhost:8888.

docker compose up --build
Demo mode
Enable demo safeguards + auto-reset for public test instances. The login page will display demo credentials.

What it is

  • FastAPI backend + server-rendered HTML templates
  • SQLite database on a Docker volume
  • Background jobs for reminders, demo resets, and purging
  • Notification channels: browser (SSE), email, webhook, Gotify, ntfy, WNS

Core features

TimeboardApp focuses on repeatable work. Recurrence is first-class, and the app is designed to stay simple to deploy and maintain.

Task board + calendar

Dashboard list view for day-to-day work and a calendar view for schedule clarity and planning.

Admin controls

User management, database export/import, email settings, logging retention, and operational utilities.

Notifications

Per-user notification services and tags. Delivery is asynchronous and events are auditable via the API.
